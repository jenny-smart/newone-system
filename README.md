# 🍋 Lemon User Import Tool

Streamlit 工具，用於從 Google Sheet 匯入「專員資料」到檸檬家事後台系統。

---

## 🚀 功能

- 從 Google Sheet 讀取資料
- 可選擇匯入「起始列 / 結束列」
- 自動轉換資料格式（生日、日期等）
- 自動登入後台並建立使用者
- 顯示匯入結果（成功 / 失敗）

---

## 📊 資料來源

- Google Sheet
- 工作表：`新人基本資料`

---
